# 学院通知看板普通用户使用手册

本手册面向使用访问验证码 `nuaa16` 进入系统的老师。普通用户可以查看通知、搜索筛选、关注个人提醒、备份提醒，并将提醒同步到手机或电脑日历；不能新增、编辑或删除通知。

## 1. 登录与进入看板

1. 打开学院通知看板链接。
2. 在“访问验证码”输入框中输入 `nuaa16`。
3. 点击“验证”进入通知看板。

进入后可以看到当前有效通知、已过期通知数量、搜索栏、分类筛选和通知卡片。

## 2. 查看通知

通知以便利贴形式展示，颜色代表通知类别，例如科研、教学、研究生、学工、安全、国合等。

每条通知通常包含：

- 标题和类别
- 发布时间
- 截止日期或倒计时
- 负责人
- 关键信息
- 附件或链接

如果想看已过期通知，勾选搜索栏旁边的“显示过期”。

## 3. 搜索与筛选

- 搜索：在搜索框输入关键词，可按标题、正文、负责人查找。
- 分类：点击“科研”“教学”“研究生”等分类标签，只看对应类型。
- 排序：可按发布时间、截止日期、重要性或类别排序。
- 视图：可在“便利贴视图”和“列表视图”之间切换。

## 4. 我的提醒

提醒功能用于把和自己有关的通知单独收藏起来。每位老师的选择互不影响。

### 4.1 添加提醒

1. 找到需要关注的通知。
2. 点击通知卡片上的“提醒我”。
3. 页面顶部“我的提醒”区域会显示这条通知。

再次点击“已提醒”或“取消”可以移除该提醒。

### 4.2 查看更多提醒

“我的提醒”默认显示前 6 条。如果超过 6 条，点击“加载更多”继续展开。

### 4.3 过期提醒

通知过期后不会自动删除，会在“我的提醒”中变灰并标记为已过期。可以点击“清除过期”一次性移除过期提醒，也可以保留用于回看。

### 4.4 备份我的提醒

点击“备份我的提醒”会复制一个备份链接。换手机、清理微信缓存或更换浏览器后，打开这个链接可以恢复“我的提醒”。

注意：备份链接不是日历地址，不能导入手机日历。

## 5. 日历订阅

如果希望系统日历自动提醒截止日期，可以点击“复制日历订阅地址”，再添加到手机或电脑日历。订阅成功后，以后在页面上新增“提醒我”，同一个日历订阅会自动同步新的提醒。

日历刷新不是实时的，具体更新时间由手机或日历软件决定，可能需要几分钟到数小时。

### 5.1 iPhone / iPad

1. 在看板中点击“复制日历订阅地址”。
2. 打开“设置”。
3. 进入“日历”→“账户”→“添加账户”→“其他”。
4. 选择“添加已订阅的日历”。
5. 粘贴订阅地址并保存。

### 5.2 Android

推荐使用 Google 日历或手机厂商自带日历的“通过网址添加日历/订阅日历”功能。

常见做法：

1. 在看板中点击“复制日历订阅地址”。
2. 打开 Google 日历网页版或支持 URL 订阅的日历应用。
3. 选择“添加日历”→“通过网址”。
4. 粘贴订阅地址并保存。
5. 等待手机日历同步。

不同安卓品牌的菜单名称可能不同，如果手机自带日历不支持 URL 订阅，可优先使用 Google 日历或 Outlook。

### 5.3 Windows

可使用 Outlook 或 Windows 日历类应用。

常见做法：

1. 在看板中点击“复制日历订阅地址”。
2. 打开 Outlook 日历。
3. 选择“添加日历”或“订阅日历”。
4. 选择“从 Web / 通过 URL”。
5. 粘贴订阅地址并保存。

### 5.4 Mac

1. 在看板中点击“复制日历订阅地址”。
2. 打开 Mac 自带“日历”应用。
3. 菜单栏选择“文件”→“新建日历订阅”。
4. 粘贴订阅地址。
5. 设置刷新频率并确认。

## 6. 微信中使用的注意事项

大多数老师会从微信里打开看板。微信内可以正常查看通知、添加“我的提醒”和复制链接。

但微信内置浏览器不一定能直接添加日历。如果点击日历相关链接后提示“在浏览器打开”，请点击微信右上角“…”并选择“在浏览器打开”，再添加到系统日历。

## 7. 常见问题 Q&A

### Q1: 输入 `nuaa16` 后能做什么？

可以查看通知、搜索筛选、查看附件、添加个人提醒、备份提醒、复制日历订阅地址。不能管理通知。

### Q2: “提醒我”会发微信消息给我吗？

不会。它是看板内的个人提醒收藏。若已订阅日历，截止日期会进入手机或电脑日历，由日历软件提醒。

### Q3: 为什么我换手机后“我的提醒”没了？

“我的提醒”默认保存在当前手机或浏览器中。换手机前请点击“备份我的提醒”，在新手机上打开备份链接即可恢复。

### Q4: 日历订阅后，新点的“提醒我”会自动进入日历吗？

会。只要使用的是“复制日历订阅地址”生成的固定订阅地址，后续新增提醒会同步到同一个日历订阅中。但日历软件刷新可能有延迟。

### Q5: 为什么日历里没有马上出现新提醒？

日历订阅由手机或日历软件定时刷新，不是实时刷新。可以等待一段时间，或在日历应用中手动刷新。

### Q6: 已过期提醒会怎样？

过期提醒会保留并变灰，方便回看。可以点击“清除过期”移除。

### Q7: 清理微信缓存会影响提醒吗？

可能会。清理缓存可能删除本机保存的提醒。建议定期使用“备份我的提醒”保存恢复链接。

### Q8: 为什么有些通知没有“提醒我”？

没有截止日期的通知不能加入截止提醒。可以通过搜索或分类再次查找。

### Q9: 备份链接和日历订阅地址有什么区别？

备份链接用于恢复“我的提醒”；日历订阅地址用于添加到日历。两者不能混用。

### Q10: 我不想继续订阅日历怎么办？

在手机或电脑日历应用中找到该订阅日历，选择删除或取消订阅即可，不会影响看板里的“我的提醒”。
