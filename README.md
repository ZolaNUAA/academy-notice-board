# 学院通知便利贴看板

一个便利贴风格的学院通知展示系统，支持普通用户查看和管理员管理。

## 启动

```bash
cd /home/zhaozola/academy-notice-board
node server.js
```

然后打开 http://localhost:3000

## 功能

- **便利贴视图**：瀑布流展示，每张便利贴带随机微旋转
- **分类筛选**：科研、教学、研究生、学工、保密、国资、安全、国合、全院等
- **重要性标识**：★★★ 高 / ★★☆ 中 / ★☆☆ 低
- **DDL倒计时**：显示距截止日期的相对时间
- **过期折叠**：过期通知默认折叠，可展开查看
- **搜索**：支持标题、正文、负责人搜索
- **管理员模式**：输入密码进入，可增删改通知
- **微信文本解析**：粘贴群通知自动分类提取

## 页面说明

1. **登录页**：输入管理员密码（或留空游客访问）
2. **便利贴视图**：普通用户看到的瀑布流看板
3. **列表管理**：管理员可对通知进行增删改操作

## 通知类型

| 类型 | 颜色 | 说明 |
|------|------|------|
| 科研 | 黄色 | 基金、申报、学术等 |
| 教学 | 橙色 | 本科、毕设、课程等 |
| 研究生 | 蓝色 | 硕士、博士、导师等 |
| 学工 | 绿色 | 辅导员、奖助等 |
| 保密 | 粉色 | 涉密相关内容 |
| 国资 | 紫色 | 资产设备采购 |
| 安全 | 浅黄 | 实验室安全培训 |
| 国合 | 青色 | 国际化、海外 |
| 全院 | 灰色 | 大会、通知 |
