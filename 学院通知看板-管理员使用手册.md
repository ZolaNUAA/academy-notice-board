## 1 系统简介

学院通知便利贴看板是一个轻量级的通知管理系统，专为高校学院日常通知发布设计。

### 核心特点

| 功能 | 说明 |
|------|------|
| 📝 微信群文本解析 | 粘贴微信群通知，自动分类、提取截止日期和负责人 |
| 🏷️ 智能分类 | 根据内容自动识别通知类型（科研、教学、研究生等） |
| 📎 附件支持 | 支持上传 PDF、Word、Excel、图片等文件 |
| 🖼️ 图片粘贴 | 直接从剪贴板粘贴图片到编辑框 |
| ⏰ 截止日期提示 | 自动识别时间信息，过期通知自动折叠 |
| ⭐ 重要性标识 | 自动判断通知的重要程度（1-3 星） |

## 2 快速入门

> 🎯 目标：5 分钟内发布第一条通知

1. **打开浏览器** — 访问管理员提供的网址

## 4 添加通知

系统支持两种方式添加通知：

### 方式一：粘贴微信群文本（推荐）

复制微信群中的通知内容，粘贴到编辑框，系统会自动解析分类。

### 方式二：手动填写（通过编辑面板）

在解析结果的编辑面板上，手动修改标题、类型、正文等信息。

### 自动解析功能说明

系统会自动识别以下信息：

| 信息 | 说明 |
|------|------|
| **通知类型** | 根据关键词自动匹配（科研、教学、国合、安全等） |
| **标题** | 自动提取【标题】或首行内容作为标题 |
| **截止日期** | 识别「截止日期」「请于 X 月 X 日前」等时间信息 |
| **重要性** | 检测「重要」「紧急」「务必」等关键词，自动评星 |
| **负责人** | 识别「联系人：XXX」「负责人：XXX」等信息 |
| **链接** | 自动提取通知正文中的 URL 地址 |

### 微信群粘贴示例

以下是一段典型的微信群通知，复制后直接粘贴到编辑框即可：

```
【国合】关于申报2026年度国际项目的通知
各位老师好，2026年度国际化项目申报工作已开始。
截止日期：4月28日下班前
联系人：张老师，邮箱 jxb_16@nuaa.edu.cn
详情见：https://cs.nuaa.edu.cn/xxx
```

粘贴后，系统会自动解析为：

| 字段 | 解析结果 |
|------|----------|
| **标题** | 【国合】关于申报2026年度国际项目的通知 |
| **类型** | 国合 · 重要性：★★★ · 截止：2026-04-28 |
| **负责人** | 邮箱 jxb_16@nuaa.edu.cn |
| **正文** | 各位老师好，2026年度国际化项目申报工作已开始... |

> 💡 **批量发布**
> 多条通知可以用【】分隔，一次粘贴即可批量发布多条通知。
> 例如：同时粘贴【科研】通知和【教学】通知，系统会分别解析。

## 5 上传附件与图片

### 上传附件

支持的文件类型：

- 📎 PDF (.pdf)
- 📎 Word (.doc/.docx)
- 📎 Excel (.xls/.xlsx)
- 📎 图片 (.jpg/.png/.gif)
- 📎 压缩包 (.zip/.rar)

上传步骤：

1. 点击「选择文件」按钮 — 在发布界面的附件区域
2. 选取本地文件 — 从电脑中选择要上传的文件
3. 自动上传 — 选中的文件会自动开始上传
4. 确认附件列表 — 上传成功后文件名会显示在列表中

> ⚠️ **文件大小限制**
> 单个文件最大 5MB，超出会提示错误。

### 粘贴图片

支持直接从剪贴板粘贴图片到编辑框：

- 🖼️ 复制图片（截图工具或右键复制图片）
- 🖼️ 在编辑框中按 `Ctrl+V` 粘贴
- 🖼️ 图片会自动上传并显示为 `![图片名](URL)` 格式
- 🖼️ 发布后图片会显示在通知正文中

## 6 预览与发布

在发布前，建议先点击「预览」按钮查看效果。

### 预览功能

预览可查看以下内容：

- ✅ 通知标题和分类标签
- ✅ 正文内容和格式（含高亮关键词）
- ✅ 截止日期和剩余时间
- ✅ 附件文件列表
- ✅ 重要性星级

在预览面板中可以直接编辑标题、类型、正文等信息。

### 发布确认

确认预览效果无误后，点击「确认发布」按钮。

发布成功后，通知会立即显示在看板首页。

> 💡 **温馨提示**
> 建议重要通知先预览再发布，确保自动解析结果准确。
> 如果分类不准确，可以在预览面板中手动调整。

## 7 管理通知

管理员登录后，可以在通知列表中对每条通知进行操作。

### 编辑

修改通知的标题、类型、正文、截止日期等信息。

### 删除

删除不再需要的通知（不可恢复）。

> ⚠️ **删除不可恢复**
> 删除通知操作不可撤销，请谨慎操作。
> 建议在删除前确认通知已不再需要。

### 过期通知管理

- 超过截止日期的通知会自动标记为「已过期」
- 过期通知在首页默认折叠显示，点击可展开查看
- 管理员可以在设置中选择显示或隐藏过期通知

## 8 管理员管理

只有超级管理员（root）可以管理其他管理员账号。

### 账号类型

| 角色 | 说明 | 权限 |
|------|------|------|
| admin | 普通管理员 | 发布、编辑、删除通知 |
| root | 超级管理员 | 所有权限 + 管理管理员账号 + 系统设置 |

### 添加管理员（root 专属）

1. 以 root 密码登录系统
2. 进入「设置」→「管理员管理」
3. 输入新管理员的用户名和初始密码
4. 新管理员可使用初始密码登录后自行修改密码

> ⚠️ **密码安全要求**
> 管理员密码必须符合安全要求：
> 长度 8 位以上，包含大写字母、小写字母、数字和特殊字符。

## 10 常见问题与技巧

### 常见问题

**Q: 为什么通知自动分类不准确？**

A: 可以手动调整。在预览面板中，点击类型标签即可切换为正确的分类。

**Q: 上传附件失败怎么办？**

A: 检查文件是否超过 5MB 限制。如果超过，请压缩后上传。同时检查网络连接是否正常。

**Q: 如何让通知显示在更前面？**

A: 系统按重要性排序（3星 > 2星 > 1星）。在正文中使用「重要」「紧急」等关键词会被自动识别为高重要性。

**Q: 粘贴后没有解析出截止日期？**

A: 确保日期格式清晰。推荐格式：「截止日期：4月28日」或「请于4月28日下班前提交」。

**Q: 通知发布后可以修改吗？**

A: 可以。在通知卡片上点击「编辑」按钮即可修改所有内容。修改后会自动重新计算过期状态。

**Q: 超过截止日期的通知会自动消失吗？**

A: 不会消失，但会自动折叠显示。用户仍然可以点击展开查看过期通知，管理员可以选择手动删除。
