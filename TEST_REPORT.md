## API 测试

```bash
# GET 获取所有通知
curl http://localhost:3000/api/notices

# POST 添加通知
curl -X POST http://localhost:3000/api/notices \
  -H "Content-Type: application/json" \
  -d '{"text":"【测试】通知内容"}'

# DELETE 删除通知
curl -X DELETE http://localhost:3000/api/notices/{id}

# PATCH 更新通知
curl -X PATCH http://localhost:3000/api/notices/{id} \
  -H "Content-Type: application/json" \
  -d '{"type":"人事","importance":3}'
```
